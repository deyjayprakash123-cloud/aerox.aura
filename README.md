# 🌌 AEROX.AURA
## *Your Data has an Atmosphere. Let it Vibe.*

**AEROX.AURA** is an ultra-smooth, procedural 3D engine designed to transform abstract GitHub repositories and complex JSON structures into a visceral, navigable "Monolith City." Built with a **Cyber-Minimalist** aesthetic, it turns "boring" technical debt into a cinematic visual journey.

## 🛠️ The "Crazy" Logic (Data-to-3D Mapping)
The world is generated in real-time based on the "DNA" of your code. Every monolith in the city is a direct reflection of a file:

* **# Depth (Height / Y-Axis):** The position of the file in the project's folder tree. Deeper files create towering monoliths that pierce the dark obsidian sky.
* **# LOC (Width / Scale):** **Lines of Code.** Massive files become wide, imposing structures; sleek, optimized scripts remain thin and elegant.
* **# Complexity (The Glitch Trigger):** Measures cyclomatic paths. High complexity triggers a custom **GLSL Glitch Shader**, causing the tower to physically jitter and fragment.
* **# Health (The Aura Scalar):** A percentage-based integrity score.
    * **Healthy (>80%):** Stable, glowing Cyan glass towers.
    * **Technical Debt (<30%):** Flickering Red emergency flares with high-frequency chromatic aberration.
* **# Status:** A final classification (DEBT, MODERATE, or HEALTHY) displayed via a glassmorphic HUD.

---

## 🌊 The Aesthetic ("Aqua-Vibe")
* **# UI/UX:** A "Liquid Bento" grid with `30px` backdrop-blur glassmorphism.
* **# Terrain:** A procedural dark-matte floor utilizing Perlin noise for realistic undulating geometry:
    $$Z = \text{Perlin}(x \cdot \text{freq}, y \cdot \text{freq}) \times \text{DataAmplitude}$$
* **# Cinematics:** Unreal Bloom, Exponential Fog, and Tilt-Shift Depth of Field for a premium "iPhone 15 Pro night mode" feel.
* **# Interaction:** Ultra-smooth **Scroll-to-Fly** navigation and mouse-tracking liquid ripple effects.

## 💻 Tech Stack

* **Animation:** [Framer Motion](https://www.framer.com/motion/) + [GSAP](https://greensock.com/gsap/)
* **Post-Processing:** `@react-three/postprocessing`
* **Deployment:** [Vercel](https://vercel.com/) (Edge Optimized)
